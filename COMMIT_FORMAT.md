# Commit Message Format

Use this format for git commit messages:

1. Title line summarizes the high-level change.
2. Keep the title about 10 words or less.
3. Follow with a few bullet points.
4. Keep each bullet about 10 words or less.

Example:

```
Refine clash logging for combat turns

- Add per-turn attack and defense breakdown
- Show armor and projected damage values
- Cover logging output with tests
```
